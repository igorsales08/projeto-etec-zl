# 🎓 Site ETEC Zona Leste - Sistema Acadêmico

Projeto full-stack desenvolvido em **Laravel** como requisito de avaliação acadêmica. O sistema contempla um portal informativo da instituição com interface moderna e uma área restrita (Back-End) para lançamento de notas, protegida por autenticação.

**Autor:** Igor Sales Moreira  
**Turma:** 3º DS AMS (Desenvolvimento de Sistemas)  

---

## 🚀 Tecnologias e Ferramentas
- **PHP 8 & Laravel 11** (Framework Core)
- **Laravel Breeze** (Sistema de Autenticação e Rotas Protegidas)
- **Tailwind CSS** (Estilização e Responsividade das Views)
- **MySQL** (Banco de Dados Relacional)
- **Blade** (Template Engine)

---

## ✅ Requisitos do Escopo Atendidos
- [x] Framework Laravel estruturado.
- [x] Views `HOME`, `CURSOS` e `EVENTOS` publicadas e estilizadas.
- [x] View `FORMULÁRIO` de notas funcional.
- [x] Autenticação (Breeze) isolando a rota do formulário.
- [x] Rota Fallback (Erro 404) customizada.
- [x] Migrations executadas (Tabela de Notas e Usuários).
- [x] Formulário com proteção de segurança `@csrf`.
- [x] Controller com validação e salvamento no banco de dados.
- [x] Código com comentários explicativos na lógica.
- [x] Repositório Git sem compactação de arquivos.
